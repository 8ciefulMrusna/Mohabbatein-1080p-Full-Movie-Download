Depuração com o avr-gdb (ATMEGA)

Com o suporte a depuração habilitado é possível utilizar o avr-gdb para depurar o código utilizado no simulador. Use a janela de configurações para escolher entre o MDB (MPLABX) e o GDB para depuração dos microcontroladores AVR.

Utilize o avr-gdb com o arquivo .elf como parâmetro: 

 avr-gdb arquivo_compilado.elf
e o comando abaixo para se conectar: 
 target remote localhost:1234

O debug no modo gráfico pode ser feito utilizando a IDE eclipse junto com o Sloeber Arduino plugin.
