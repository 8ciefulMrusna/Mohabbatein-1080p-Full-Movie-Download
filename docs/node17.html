Depuração Integrada com o MPLABX (PIC e ATMEGA)

Para utilizar o IDE MPLABX para depurar e programar o PicsimLab, basta instalar o plugin com-picsim-picsimlab.nbm no MPLABX.

O plugin se conecta ao Picsimlab através de um socket TCP na porta 1234 (ou outra definida na janela de configuração), permita o acesso no firewall.
