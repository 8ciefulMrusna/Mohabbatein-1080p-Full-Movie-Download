Help

Commands

  • Click in ICSP connector to load an .hex file.
  • Click in PWR button to ON/OFF the emulator..
  • The buttons can be activated through mouse or keys 1, 2, 3 e 4.

Features of board 1

Emulates the Labtools development board McLab1 who uses one PIC16F628A.

McLab1 kit buy, manual and examples in download area www.mosaico.com.br

The hardware and the use of kit is described in the book Desbravando o PIC - Ampliado e Atualizado para PIC 16F628A of Erica publisher (ISBN: 978-85-7194-867-9).

Features of board 2

Emulates an didatic board developed by author.

Features of boards 3

Emulates the Labtools development board McLab2 who uses one PIC16F877A or one PIC18F452.

McLab2 kit buy, manual and examples in download area www.mosaico.com.br

The hardware and the use of kit is described in the book Conectando o PIC - Recursos Avançados of Erica publisher (ISBN: 978-85-7194-737-5).

Features of boards 4

Emulates the microgenius development board PICGenios PIC18F e PIC16F Microchip who uses one PIC16F877A or one PIC18F452.

PICGenios PIC18F e PIC16F Microchip kit buy and manual in www.microgenios.com

Programmer connection

To use the embedded picstart+ emulator, install an NULL-MODEM emulator:

  • Windows: com0com http://sourceforge.net/projects/com0com/
  • Linux: tty0tty http://sourceforge.net/projects/tty0tty/ or
    http://github.com/lcgamboa/tty0tty

Configuration examples:

OS PicsimLab port IDE port NULL-Modem prog. Connection
Windows wport=com8 Mplab=com2 com0com com2<=>com8
Linux lport=/dev/tnt4 Piklab=/dev/tnt5 tty0tty /dev/tnt4<=>/dev/tnt5

MPLABX Integrated Debug

To use the MPLABX IDE for debug and program the PicsimLab, install the plugin com-picsim-picsimlab.nbm in MPLABX.

The plugin connect to Picsimlab through a TCP socket using port 1234, allow the access in the firewall.
